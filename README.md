# HarFileChecker
HAR（ブラウザが記録するネットワークログ）から、ページ上で発火した計測タグ（例: Google Analytics, GTM, Facebook Pixel など）を探して一覧にするシンプルなツールです。

デザイナーやマーケター向けに作られており、特別な開発環境がなくてもブラウザで使えます。

デモ: https://azuwata.github.io/HarFileChecker/

## 使い方（非エンジニア向け）
- 1. ページのネットワークをHAR形式でエクスポートします（ブラウザの開発者ツールで「保存 / Export HAR」等）。
- 2. このツールのページ（`index.html`）をブラウザで開きます。
- 3. 「HARファイルをここにドロップ」部分にファイルをドラッグするか、クリックしてHARファイルを選択します。
- 4. 検索したいタグ名やキーワードを入力（複数語はスペース区切りでAND検索）。
- 5. 「チェック開始」を押すと、発火したリクエストの一覧が表示されます。
- 6. 必要なら「CSV出力」で結果をダウンロードできます。

## 今回のセキュリティと堅牢化のポイント（要約）
- 出力のサニタイズ: HAR内の値をそのままHTMLに埋め込まず安全化しています（XSS対策）。
- HAR形式の検証: 期待する構造でないファイルは弾くようにしました（不正ファイルでのクラッシュ防止）。
- CSV出力の保護: Excelで開いたときに式として実行される危険を避けるため、先頭が `=`, `+`, `-`, `@` の場合は無害化しています。

これらの改善により、安全に社内のHARを使ってタグの発火確認ができるようになっています。

## 動作確認（簡単）
- `index.html` をブラウザで開き、サンプルのHARファイルを読み込んで通常通り動作するか確認してください。
